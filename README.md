# Book Collection App
A simple app to maintain record of book collection. It's built using serverless framework on AWS. 

Key features of the app include:
- Using this app, a reader to keep reocrd of books read and books to be read in future. 
- A short of summary of books already read could be maintained. 
- A picture of book or summary could also be added 

## Tech Stack
- AWS Lambda
- S3 Bucket
- AWS API Gateway
- DynamoDB
- Serverless Framework
